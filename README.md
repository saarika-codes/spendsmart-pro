# рџ“Њ Overview

SpendSmart Pro is a feature-rich frontend application that simulates a modern personal finance management platform. The application enables users to manage transactions, monitor monthly budgets, visualize financial analytics, and track savings goals without requiring a backend or authentication.

All application data is stored using **Local Storage**, making it lightweight while demonstrating real-world frontend architecture and state management.
